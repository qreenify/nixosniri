# Apps That Omarchy Themes

Based on the theme files in `~/git/omarchy/themes/`, omarchy provides theming for:

## Terminal Emulators
- **Alacritty** (`alacritty.toml`) - Your current terminal
- **Ghostty** (`ghostty.conf`) - Alternative terminal
- **Kitty** (`kitty.conf`) - Alternative terminal

## Desktop Environment
- **Hyprland** (`hyprland.conf`) - Window manager theme colors
- **Hyprlock** (`hyprlock.conf`) - Lock screen
- **Waybar** (`waybar.css`) - Status bar colors
- **Walker** (`walker.css`) - Application launcher (you're using this!)
- **Mako** (`mako.ini`) - Notification daemon
- **SwayOSD** (`swayosd.css`) - On-screen display for volume/brightness

## Applications
- **Neovim** (`neovim.lua`) - Text editor colorscheme
- **VS Code** (`vscode.json`) - Code editor theme
- **Chromium/Brave** (`chromium.theme`) - Browser theme
- **Btop** (`btop.theme`) - System monitor

## Visual Elements
- **Icons** (`icons.theme`) - Icon theme selection
- **Wallpapers** (`backgrounds/`) - Desktop backgrounds
- **Preview** (`preview.png`) - Theme preview image
